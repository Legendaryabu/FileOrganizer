# Python File Organizer

A practical automation tool that organizes cluttered directories by scanning file extensions and moving them into categorized folders (e.g., Images, Documents, PDFs).

## 🚀 Key Features
* **Extension-Based Sorting:** Automatically identifies file types and moves them to relevant sub-folders.
* **Path Management:** Uses Python's `os` and `shutil` modules for safe and efficient file handling.
* **Workflow Automation:** Designed to save time by cleaning up Downloads or Desktop folders instantly.

## 🛠️ Tech Stack
* **Language:** Python 3.x
* **Core Modules:** `os`, `shutil`

## 📂 How to Use
1. Run the `file_organizer.py` script.
2. Provide the directory path you wish to organize.
3. The script will automatically create folders and move files accordingly.
